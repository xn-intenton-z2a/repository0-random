# LEVENSHTEIN

Summary

Implement Levenshtein distance to compute the minimal number of single-character edits required to change one string into another. Behavior: treat null or undefined as an empty string; support Unicode code points and return an integer distance.

Acceptance Criteria

- The Levenshtein distance between "kitten" and "sitting" is 3.
- The Levenshtein distance between two empty strings is 0.
- Null or undefined inputs are treated as empty strings.
