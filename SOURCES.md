# Sources

Reference material and documentation sources for this project.

- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math — MDN Math object reference
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Function — MDN Function constructor (dynamic expression parsing)
- https://developer.mozilla.org/en-US/docs/Web/SVG/Element/polyline — MDN SVG polyline element (points attribute)
- https://developer.mozilla.org/en-US/docs/Web/SVG/Attribute/viewBox — MDN SVG viewBox attribute (scaling and coordinate system)
- https://csv.js.org/parse/ — csv-parse (Node.js CSV parser)
- https://github.com/Automattic/node-canvas — node-canvas (Canvas API for Node.js; toBuffer -> PNG)
- https://sharp.pixelplumbing.com/ — sharp (SVG to PNG conversion and image processing)
